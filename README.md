ToDoAvatar
===================


- ToDoAvatar는 안드로이드 스마트폰과 Sony사의 SmartWatch2를 연동하여 사용하는 To Do List 애플리케이션입니다. Android 4.0 이상의 디바이스 내에서 사용 가능하며, 기본 연동을 위해 스토어에서 SmartWatch2 SW2를 다운받아 설치해야 합니다.
